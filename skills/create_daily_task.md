## 字段（从任务列表页 & 详情弹窗推得）

创建一条任务/排期大概率需要：

| 字段 | 来源 | 备注 |
|---|---|---|
| 责任人 | 用户下拉 | 排期表第 1 列 |
| 部门 | 部门下拉 | 默认当前用户所在部门 |
| 项目 | 项目下拉 | 排期表第 4 个 select；带搜索；非项目任务不需要 |
| 服务类型 | 任务列表页有列 | 如 `建模`、`动画`、`平台研发` 等 |
| 任务名称 | 文本 | 如 `平台研发-前端开发、产品方案、项目管理 \| 18H`，惯例是 `<服务类型>-<细分> \| <工时>H` |
| 任务描述 | 多行文本 | |
| 产出类型 | 枚举 | `产出` / `非产出`（NPT） |
| 排产工时 | 数字（h） | |
| 开始日期 / 截至日期 | 日期 | |
| 任务状态 | 枚举 | `未开始` / `进行中` / `完工` / `搁起` 等（颜色：橙=进行中，绿=完工，深=搁起） |
