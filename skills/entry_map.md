# 51PM — 入口地图（实测确认的功能入口，全 51pm skill 共享）

> **本文件是唯一权威的入口登记处**，由验收/操作类 skill 在每轮实测后回填维护。任何 51pm skill 找功能入口时**先查这里**，查不到再现场找，找到后**必须回填**。

## 回填规则（对所有 51pm skill 生效）

1. **新确认的入口**（表里没有的）：验收/操作结束交付前，立即追加一行
2. **纠正的入口**（实测与表不符）：更新对应行，旧入口若易混淆保留 ⚠️ 备注
3. **同入口踩到的坑**（静默失败、旧按钮混淆、权限差异等）：以 ⚠️ 写进备注列
4. 入口写法统一：「一级菜单 → 二级页面 → 具体控件」，悬停触发标注 `(悬停)`
