# 产能数据看板【只读】

> 一句话：看工程/小组产能分析，点色块下钻需求列表。典型 AI 复用场景：「定时拉产能分布数据 / 盘点各组在制需求」。

## 参数

| 参数          | 示例            | 说明                                             |
| ------------- | --------------- | ------------------------------------------------ |
| `env`         | `test` / `prod` | 测试 `10.67.8.183:7777`（默认）/ 正式；只读默认复用不问 |
| `dept_id[]`   | `12` / 空       | 部门 id 数组，空=全部                            |
| `start_date`  | `2026-01-01`    | 统计起始日                                       |
| `end_date`    | `2026-12-31`    | 统计结束日                                       |
| `filter_type` | `all`           | 过滤类型                                         |

## 入口

- **路由**：`/statistic/capacity_analysis`（实测）
- **导航**：统计 → 产能数据看板；页头「工程产能分析 / 小组产能看板」切换
- **等待锚点**：`wait_for_selector('.bar-seg--clickable, .el-table', 15000)`
- **进站**：先按 [../README.md](../README.md) Tab 复用，禁止首选 new_tab

## 操作步骤（真实 UI）

> 文字版流程，先建立整体认知；自动化执行看下方「步骤」，只读定时任务优先走「可直调接口」。

1. 进产能数据看板
2. 「工程产能分析」维度看产能柱状图
3. 点色块 `.bar-seg--clickable`（`--doing/--wait/--pause`）下钻需求列表
4. 切「小组产能看板」维度

## 步骤

### 1.（推荐·定时任务）接口直调，秒出数据

```js
const { request } = require('@playwright/test');
const fs = require('fs');
const state = JSON.parse(fs.readFileSync('auth/state.json', 'utf8'));
const o = state.origins.find(x => x.origin.includes('10.67.8.183'));
const token = o.localStorage.find(l => l.name === 'oauthToken').value;
(async () => {
  const ctx = await request.newContext({
    baseURL: process.env.PM_API_BASE || 'http://localhost:8888',   // 需先 npm run proxy；可直连时设 http://10.67.8.183:8888
    extraHTTPHeaders: { Authorization: token, token },
  });
  const r = await ctx.get('/manage_api/data_export/get_employee_project_list?dept_id[]=&start_date=2026-01-01&end_date=2026-12-31&filter_type=all');
  const j = await r.json();
  console.log('code', j.code, '条数', (j.data?.list || j.data || []).length);   // code 必须 0
  await ctx.dispose();
})();
```

### 2.（UI 复现）切维度、点色块下钻

```js
// 切维度：按页头文本点击
Array.from(document.querySelectorAll(".el-radio-button, .tab, span, div"))

// 点色块下钻需求列表
document.querySelector(".bar-seg--clickable")?.click();   // 类名含 --doing/--wait/--pause
```

## 可直调接口（免 UI 快路径 —— 定时/只读优先）

- `GET /manage_api/data_export/get_employee_project_list?dept_id[]=&start_date=&end_date=&filter_type=all`
  - 返回取值：`j.data`（list/数组）；`j.code` 必须为 0
  - 边界：非法日期静默兜底不报错；不存在部门返空数组（断言只验不 5xx）
- ⚠️ 非生产成员过滤名单刷新恢复默认
