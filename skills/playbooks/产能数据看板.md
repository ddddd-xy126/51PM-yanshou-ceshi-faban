# 产能数据看板【只读】

> 一句话：看工程/小组产能分析，点色块下钻需求列表。

- **入口**：统计 → 产能数据看板；页头「工程产能分析 / 小组产能看板」切换
- **路由**：`/statistic/capacity_analysis`
- **适用/触发**：查产能分布、下钻需求

## 操作步骤（真实 UI）

1. 进产能数据看板
2. 「工程产能分析」维度看产能柱状图
3. 点色块 `.bar-seg--clickable`（`--doing/--wait/--pause`）下钻需求列表
4. 切「小组产能看板」维度

## 可直调接口

- `data_export/get_employee_project_list?dept_id[]=&start_date=&end_date=&filter_type=all`
  - 边界：非法日期静默兜底不报错；不存在部门返空数组（断言只验不 5xx）
- ⚠️ 非生产成员过滤名单刷新恢复默认

## 已知坑

- ⚠️ 下钻需求数应与色块数值一致（数据断言点）
