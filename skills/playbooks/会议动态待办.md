# 会议动态待办（负责人 + 状态）【写】

> 一句话：在项目动态里添加会议动态，其待办事项每条可设负责人与状态。

- **入口**：项目 → 项目详情 → 项目动态 → 添加动态（会议动态）→ 待办事项
- **适用/触发**：会议纪要跟进待办，指定负责人与进度

## 操作步骤（真实 UI）

1. 进项目动态页 → 添加动态（会议动态）
2. 待办事项区添加条目
3. 每条待办设：状态（待处理/进行中/已完成）+ 相关人员
   - ⚠️ 相关人员 el-select 要点 `.el-select__input.is-mini`（tags 拦截坑，同组群弹窗）
4. 保存 → 待办状态与人员正确回读

## 可直调接口

- `project_moment/get_list?project_id=N&type=1`（type=1 会议动态，返回待办含负责人 + 状态字段，可作落库断言）

## 已知坑

- ⚠️ 相关人员 el-select 需点 `.el-select__input.is-mini`（步骤 3）
