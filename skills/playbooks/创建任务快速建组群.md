# 创建任务时快速建组群并导入成员【写】

> 一句话：在创建任务选指派人的「从组群导入」里就地新建组群、加人、保存，随即导入使用，组群顺手沉淀下次复用。

- **入口**：创建多人通用任务 / 需求拆解向导 → 「从组群导入」popover → 「立即管理」链接 → 就地弹「组群配置」弹窗
- **适用/触发**：批量指派任务时顺手沉淀常用人员组群

## 操作步骤（真实 UI）

1. 进入创建任务向导（多人通用任务 / 需求拆解），到指派人环节点「从组群导入」popover
2. popover 内点「立即管理」→ **就地弹出「组群配置」弹窗**（不离开创建流程）
3. 「新建分组」→ 填组名 → 创建成功（自动分配 ID 并选中）
4. 成员下拉真实点击选人 → 「加入」→ 「保存成员」→ toast「保存成功」
   - ⚠️ el-select 多选外层 input 会被 tags 里 mini input 拦截，要点 `.el-select__input.is-mini` / `.is-small` 才有效
5. 关闭组群配置回到创建流程 → 再点「从组群导入」→ 组群下拉出现新组群 → 选中「添加」→ toast「已添加组群『xxx』成员 N 人」，指派人自动填入

## 已知坑

- ⚠️ 加人是**两步**：「加入」（加进待保存）+「保存成员」（落库），都要点
- ⚠️ el-select 多选 input 被 tags mini input 拦截（步骤 4）
