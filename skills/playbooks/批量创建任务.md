# 批量创建任务（通用批量表单向导）【写】

> 一句话：用两步向导 + 侧栏条目管理一次创建多条任务（独立任务=每条字段各异；多人通用=编辑一次对全部指派人生效）。

- **入口**：非项目 → 需求 → 点需求名进任务页 → 左上「创建任务」(悬停) → 独立任务 / 多人通用任务
- **路由**：`/not_project/not_project_task?demandId=N`
- **适用/触发**：给一个需求下批量建任务；多人会议/培训任务批量指派

## 操作步骤（真实 UI）

1. 进入需求上下文（**必须带 demandId**，否则提交静默失败）：非项目 → 需求列表 → 点需求名进任务页
2. 左上「创建任务」(悬停展开) → 选「独立任务」或「多人通用任务」
3. 通用批量表单向导（BatchCreateDialog）两步：
   - 第 1 步录入条目：侧栏显示「共 N 条 / 有效 N / 全清」，空白条目自动丢弃
   - 独立任务：每条字段可各不相同；多人通用：编辑一次对全部指派人生效
4. 第 2 步确认 → 提交，toast「成功创建 N 条任务」

## 已知坑

- ⚠️ **无需求上下文直接提交静默失败**（后端 code 51 不透出）——务必从需求进入
- ⚠️ 需求列表行内第一个下拉只有「F自由需求」（那是建子需求，不是建任务）
- ⚠️ 非项目任务表 console 有 `Duplicate keys pause` 红错（notTaskTable.vue，已知无害）
