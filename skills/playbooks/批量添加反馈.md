# 批量添加项目反馈（通用批量表单向导）【写】

> 一句话：用两步向导批量提交项目反馈申请，选一次项目对全部条目生效。

- **入口**：我的地盘 → 我的反馈 → 添加反馈 → 批量添加反馈（「批量添加项目反馈」弹窗）
- **适用/触发**：一次性提交多条项目反馈申请

## 操作步骤（真实 UI）

1. 我的地盘 → 我的反馈 → 添加反馈 → 批量添加反馈
2. 两步向导 + 侧栏条目（共 N 条 / 有效 N / 全清），空白条目自动丢弃
3. 选项目一次对全部条目生效；逐条填商机号/模块/内容
4. 提交 → 落库

## 可直调接口

- `produce_demand/add_apply_demand`：空商机号/空模块 → code 51
