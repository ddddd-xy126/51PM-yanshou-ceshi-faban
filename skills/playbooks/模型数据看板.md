# 模型数据看板【只读】

> 一句话：看模型发包/资产的总览指标与明细，区分供应商与自制口径。典型 AI 复用场景：「定时拉模型发包/资产总览、量化供应商交付」。

## 参数

| 参数             | 示例            | 说明                                   |
| ---------------- | --------------- | -------------------------------------- |
| `env`            | `test` / `prod` | 测试（默认）/ 正式；只读默认复用不问   |
| `period`         | `month` / 空    | 统计周期；非法/空静默兜底返全量        |
| `producer_scope` | `all`           | 制作方口径（供应商/自制/全部）         |
| `score_scope`    | `all`           | 评分口径                               |

## 入口

- **路由**：`/statistic/outsource_panel`（实测）
- **导航**：统计 →「资产」分组 → 模型数据看板；页头「模型数据总览 / 模型明细」切换
- **等待锚点**：`wait_for_selector('.el-table, .overview-card', 15000)`
- **进站**：先按 [../README.md](../README.md) Tab 复用，禁止首选 new_tab

## 操作步骤（真实 UI）

> 文字版流程，先建立整体认知；自动化执行看下方「步骤」，只读定时任务优先走「可直调接口」。

1. 进总览页：发包数量/发包金额/资产数量/资产评分/不合格占比/平均反馈数六卡 + 金额、打分、类型状态、属性质量、省份热力、评分档位等图表
2. 交叉验算（如发包 272 = 供应商 150 + 自制 122；金额 376.8 万 = 261.5 + 115.3）
3. 切「模型明细」：发包维度(272) / 资产维度(316) 切换，支持自制/外包筛选

## 步骤

### 1.（推荐·定时任务）接口直调总览，秒出数据

```js
const { request } = require('@playwright/test');
const fs = require('fs');
const state = JSON.parse(fs.readFileSync('auth/state.json', 'utf8'));
const o = state.origins.find(x => x.origin.includes('10.67.8.183'));
const token = o.localStorage.find(l => l.name === 'oauthToken').value;
(async () => {
  const ctx = await request.newContext({
    baseURL: process.env.PM_API_BASE || 'http://localhost:8888',
    extraHTTPHeaders: { Authorization: token, token },
  });
  const r = await ctx.get('/manage_api/outsource/get_data_overview?period=month&producer_scope=all&score_scope=all');
  const j = await r.json();
  const ov = j.data.overview;   // ⚠️ 实测取值路径是 j.data.overview（不是 j.data.data.overview）
  console.log('发包总数', ov.package_total, '资产总数', ov.asset_total);
  await ctx.dispose();
})();
```

## 可直调接口（免 UI 快路径 —— 定时/只读优先）

- 总览 `outsource/get_data_overview?period=&producer_scope=&score_scope=`
  - 返回取值：`j.data.overview`（六卡指标）、`j.data.charts`（图表）；非法 period/空参静默兜底返全量
- 明细 `get_package_dimension_list`（非法 status 返 0 条）/ `get_asset_dimension_list`
- **一键定时脚本**：[../../regression/scripts/reports/daily-stats.js](../../regression/scripts/reports/daily-stats.js) 已含本接口总览统计

## 读数 / 断言锚点

- 六卡：发包数量 / 发包金额 / 资产数量 / 资产评分 / 不合格占比 / 平均反馈数
- 交叉验算：发包总数 = 供应商 + 自制；金额同理
