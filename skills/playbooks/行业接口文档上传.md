# 行业接口文档上传【写】

> 一句话：在项目概况右侧信息栏上传「行业接口文档」，与定制接口文档独立互不覆盖。

- **入口**：项目 → 项目列表 → 点项目名进项目概况 → 右侧项目信息栏「上传行业接口文档」
- **适用/触发**：给项目补充行业接口文档

## 操作步骤（真实 UI）

1. 进项目概况页
2. 右侧信息栏找「上传行业接口文档」（位于『定制接口文档』与旧文档项之间/下方）
3. 点上传 → 选文件后**还要点「确认上传」**（两步）
4. 上传后文档出现在信息栏，与定制接口文档互不覆盖

## 已知坑

- ⚠️ 选文件后必须点「确认上传」才落库
- ⚠️ 上传触发的 filechooser 必须消费（setFiles），不消费会挂起导致页面跳回 main
