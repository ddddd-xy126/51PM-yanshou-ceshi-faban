# 递交列表 — 跳转项目 / 超时递交筛选【只读】

> 一句话：在递交列表按状态/超时/日期筛选递交记录，点项目名可跳项目概况。典型 AI 复用场景：「定时统计超时递交、拉全年递交记录」。

## 参数

| 参数                    | 示例                | 说明                                                    |
| ----------------------- | ------------------- | ------------------------------------------------------- |
| `env`                   | `test` / `prod`     | 测试（默认）/ 正式；只读默认复用不问                    |
| `is_over_tb_time`       | `1` / `-1` / `99`   | 1→超时集、-1→全量、99→0 条；非法值不校验返部分集         |
| `keywords`              | `某项目`            | 按项目名精确匹配                                        |
| `start_date`/`end_date` | `2026-01-01`        | 日期范围（列表默认只筛今天，须放宽）                    |

## 入口

- **路由**：`/OPStestList/OPStestList_list`（实测）
- **导航**：递交 → 递交列表
- **等待锚点**：`wait_for_selector('.el-table', 15000)`
- **进站**：先按 [../README.md](../README.md) Tab 复用，禁止首选 new_tab

## 操作步骤（真实 UI）

> 文字版流程，先建立整体认知；自动化执行看下方「步骤」，只读定时任务优先走「可直调接口」。

### A. 筛选「仅查看超时递交」（V2.2.8）

1. 打开递交列表
2. 勾选筛选区「仅查看超时递交」复选框（递交状态下拉右侧）
3. ⚠️ 列表默认只筛今天 → 放宽日期范围（如全年）再查；日期 `fill` 不触发，走 vm `form` + `searchData()`
4. 读结果行的要求/实际递交时间核对

### B. 跳转项目（V2.2.3 折叠修复路径）

1. 点项目名（左固定列）→ popconfirm「确定」→ 项目概况
2. 浏览器后退返回，列表完整渲染（折叠 BUG 已修复）

## 步骤

### 1.（推荐·定时任务）接口直调超时递交，秒出数据

```js
const { request } = require('@playwright/test');
const fs = require('fs');
const state = JSON.parse(fs.readFileSync('auth/state.json', 'utf8'));
const o = state.origins.find(x => x.origin.includes('10.67.8.183'));
const token = o.localStorage.find(l => l.name === 'oauthToken').value;
(async () => {
  const ctx = await request.newContext({
    baseURL: process.env.PM_API_BASE || 'http://localhost:8888',
    extraHTTPHeaders: { Authorization: token, token },
  });
  // is_over_tb_time=1 只取超时集；日期放宽到全年（默认只筛今天）
  const r = await ctx.get('/manage_api/project_publish/get_list?is_over_tb_time=1&start_date=2026-01-01&end_date=2026-12-31');
  const j = await r.json();
  console.log('超时递交条数', (j.data?.list || j.data || []).length);
  await ctx.dispose();
})();
```

## 可直调接口（免 UI 快路径 —— 定时/只读优先）

- `project_publish/get_list?...&is_over_tb_time={v}`：`1`→超时集、`-1`→全量、`99`→0 条
  - 返回取值：`j.data`（list/数组）
  - ⚠️ 非法值 `abc` 后端**不校验**返部分集（介于两者之间）；checkbox true-value=1 / false-value=-1
- `keywords` 参数按项目名精确匹配
- **一键定时脚本**：[../../regression/scripts/reports/daily-stats.js](../../regression/scripts/reports/daily-stats.js) 已含超时递交统计
