# 项目人员看板【只读】

> 一句话：按六维度看成员承接项目分布与状态，「项目场景」维度可下钻组员看板。典型 AI 复用场景：「定时盘点各组项目推进 / 拉人员承接分布数据」。

## 参数

| 参数         | 示例                  | 说明                                                        |
| ------------ | --------------------- | ----------------------------------------------------------- |
| `env`        | `test` / `prod`       | 测试 `10.67.8.183:7777`（默认）/ 正式；只读，默认复用不问    |
| `type`       | `pm` / `dta` / `tech` | 维度：项目管理 / DTA资产 / 项目技术（接口参数，非法/空→code 51） |
| `start_year` | `2025`                | 开工统计起始年                                              |
| `end_year`   | `2026`                | 开工统计结束年                                              |

## 入口

- **路由**：`/statistic/pm_panel`（实测；拼到 env host）
- **导航**：统计 →「管理」分组 → 项目人员看板；页头六维度切换（项目管理(含PM/QA) / 项目场景 / 项目开发 / Web端开发 / 项目设计 / 项目技术 / DTA资产）
- **等待锚点**：`wait_for_selector('.psb-name, .el-table', 15000)`
- **进站**：先按 [../README.md](../README.md) Tab 复用，禁止首选 new_tab

## 操作步骤（真实 UI）

> 文字版流程，给人/AI 先建立整体认知；要真正自动化执行看下方「步骤」的可执行片段，只读定时任务优先走「可直调接口」。

1. 进看板，页头六维度切换：项目管理(含 PM/QA) / 项目场景 / 项目开发 / Web端开发 / 项目设计 / 项目技术 / DTA资产
2. 「项目场景」维度点场景名（`.psb-name.is-clickable`）就地展开「场景 · 组员看板」
3. 每张人员卡看六指标：25-26年开工 / 7天活跃 / 超周无进展 / 制作中 / 已营收 / 未开工
4. 可按开工年份、交付状态筛选；0 项目的人员卡为正常空态（指标显示 `-`）

## 步骤

### 1.（推荐·定时任务）接口直调，秒出数据

只读统计**优先走接口**，不必开浏览器。可直接 `node` 跑，token 从 `auth/state.json` 读：

```js
// 复用 regression/scripts/reports/daily-stats.js 的模式；单独跑某维度：
const { request } = require('@playwright/test');
const fs = require('fs');
const state = JSON.parse(fs.readFileSync('auth/state.json', 'utf8'));
const o = state.origins.find(x => x.origin.includes('10.67.8.183'));
const token = o.localStorage.find(l => l.name === 'oauthToken').value;
(async () => {
  const ctx = await request.newContext({
    baseURL: process.env.PM_API_BASE || 'http://localhost:8888',   // 需先 npm run proxy；可直连后端时设 http://10.67.8.183:8888
    extraHTTPHeaders: { Authorization: token, token },
  });
  for (const type of ['pm', 'dta', 'tech']) {
    const r = await ctx.get(`/manage_api/data_export/get_user_project?start_year=2025&end_year=2026&ecp_kaigong_zhuangtai=&type=${type}`);
    const j = await r.json();
    const d = j.data; const rows = Array.isArray(d) ? d : (d.data || d.list || []);
    console.log(type, '组/人数:', rows.length);   // code 必须为 0；非 0 = token 过期 → npm run login
  }
  await ctx.dispose();
})();
```

## 可直调接口（免 UI 快路径 —— 定时/只读优先）

- `GET /manage_api/data_export/get_user_project?start_year=&end_year=&ecp_kaigong_zhuangtai=&type={pm|dta|tech}`
  - 返回取值：`j.data`（数组或含 `data`/`list`）；`j.code` 必须为 0
  - 边界：非法/空 `type` → `code 51`；「项目场景」维度取数走 tab 内缓存通道，不单独发请求
- **一键定时脚本**：[../../regression/scripts/reports/daily-stats.js](../../regression/scripts/reports/daily-stats.js) 已含本接口三维度统计 + 递交超时 + 模型看板，`node scripts/reports/daily-stats.js` 落盘归档

- ⚠️ 接口走 `localhost:8888` 需先 `npm run proxy`；无人值守可直连后端 `PM_API_BASE=http://10.67.8.183:8888`（免 proxy，视网络可达性）
