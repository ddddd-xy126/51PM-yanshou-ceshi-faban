# 项目人员看板【只读】

> 一句话：按六维度看成员承接项目分布与状态，「项目场景」维度可下钻组员看板。

- **入口**：统计 → 「管理」分组 → 项目人员看板
- **路由**：`/statistic/pm_panel`
- **适用/触发**：按人员盘点各组项目推进、承接分布

## 操作步骤（真实 UI）

1. 进看板，页头六维度切换：项目管理(含PM/QA) / 项目场景 / 项目开发 / Web端开发 / 项目设计 / 项目技术 / DTA资产
2. 「项目场景」维度点场景名（`.psb-name.is-clickable`）就地展开「场景 · 组员看板」
3. 人员卡六指标：25-26年开工 / 7天活跃 / 超周无进展 / 制作中 / 已营收 / 未开工
4. 支持按开工年份与交付状态筛选

## 可直调接口

- `data_export/get_user_project?start_year=&end_year=&type={pm|dta|tech}`
  - 边界：非法/空 type → code 51；项目场景维度取数走 tab 内缓存通道，不单独发请求
