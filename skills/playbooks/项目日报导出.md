# 项目日报导出（含制作截图选项）【只读/导出】

> 一句话：导出某项目的项目日报 Excel，可选时间范围与「包含制作截图」。典型 AI 复用场景：「按项目定时导出日报 Excel 归档」。

## 参数

| 参数             | 示例            | 说明                                        |
| ---------------- | --------------- | ------------------------------------------- |
| `env`            | `test` / `prod` | 测试（默认）/ 正式；只读默认复用不问        |
| `project_id`     | `6690`          | 目标项目 id（空 → code 51）                 |
| `start_date`     | `2026-07-01`    | 起始日（不选=全部）                         |
| `end_date`       | `2026-07-31`    | 结束日                                      |
| `include_images` | `true`/`false`  | 是否含制作截图（非法值按 false）            |

## 入口

- **入口**：项目 → 项目列表 → 行内「查看详情」→「项目日报」弹窗 → 右上「导出」→「导出项目日报」配置弹窗
- **路由**：无独立路由（走项目列表行内详情弹窗）
- **等待锚点**：`wait_for_selector('.el-dialog', 15000)`
- **进站**：先按 [../README.md](../README.md) Tab 复用，禁止首选 new_tab

## 操作步骤（真实 UI）

> 文字版流程，先建立整体认知；自动化执行看下方「步骤」，只读定时任务优先走「可直调接口」。

1. 项目列表行内「查看详情」→ 打开「项目日报」弹窗
2. 右上「导出」→ 弹「导出项目日报」配置弹窗
3. 设时间范围（不选=全部）+ 拨「包含制作截图」el-switch（勾选后有「文件可能较大」告警）
4. 确认导出

## 步骤

### 1.（推荐·定时任务）接口直调导出，直接落 xlsx

```js
const { request } = require('@playwright/test');
const fs = require('fs');
const state = JSON.parse(fs.readFileSync('auth/state.json', 'utf8'));
const o = state.origins.find(x => x.origin.includes('10.67.8.183'));
const token = o.localStorage.find(l => l.name === 'oauthToken').value;
(async () => {
  const ctx = await request.newContext({
    baseURL: process.env.PM_API_BASE || 'http://localhost:8888',
    extraHTTPHeaders: { Authorization: token, token },
  });
  const r = await ctx.get('/manage_api/project_task_estimate/export_estimate_list_by_project_id?project_id=6690&start_date=&end_date=&include_images=false');
  fs.writeFileSync('日报.xlsx', await r.body());   // 空 project_id → code 51；不存在 id → 空模板 xlsx
  await ctx.dispose();
})();
```

### 2.（UI 复现）配置弹窗导出

```js
// 项目列表行「查看详情」→「项目日报」弹窗 → 右上「导出」
// 「包含制作截图」是 el-switch：
document.querySelector(".el-dialog .el-switch")?.click();   // 勾选后有「文件可能较大」告警
```

## 可直调接口（免 UI 快路径 —— 定时/导出优先）

- `project_task_estimate/export_estimate_list_by_project_id?project_id=&start_date=&end_date=&include_images=`
  - 返回取值：响应体即 xlsx 二进制（`await r.body()`）
  - 边界：空 `project_id` → code 51；不存在 id → 空模板 xlsx；非法 `include_images` → 按 false 处理

## 读数 / 断言锚点

- 「包含制作截图」= `.el-dialog .el-switch`
- 导出成功判定：响应为 xlsx 二进制（非 JSON 错误体）
