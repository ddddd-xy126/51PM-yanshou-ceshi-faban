# 项目日报导出（含制作截图选项）【只读/导出】

> 一句话：导出某项目的项目日报 Excel，可选时间范围与「包含制作截图」。

- **入口**：项目 → 项目列表 → 行内「查看详情」→「项目日报」弹窗 → 右上「导出」→「导出项目日报」配置弹窗
- **适用/触发**：按项目导日报；需要带制作截图的日报

## 操作步骤（真实 UI）

1. 项目列表行内「查看详情」→ 打开「项目日报」弹窗
2. 右上「导出」→ 弹「导出项目日报」配置弹窗
3. 设时间范围（不选=全部）+ 拨「包含制作截图」el-switch（勾选后有「文件可能较大」告警）
4. 确认导出

## 可直调接口

- `project_task_estimate/export_estimate_list_by_project_id?project_id=&start_date=&end_date=&include_images=`
  - 边界：空 `project_id` → code 51；不存在 id → 空模板 xlsx；非法 `include_images` → 按 false 处理
