# 项目递交模块（项目内递交记录）【只读】

> 一句话：在项目详情内查看该项目全部递交记录与按时率统计，支持排序。

- **入口**：项目 → 项目详情 → 二级菜单「项目递交」
- **路由**：`/project/project_publish?projectId=N`
- **适用/触发**：查单个项目的递交历史与按时率

## 操作步骤（真实 UI）

1. 进项目详情，点二级菜单「项目递交」
2. 顶部统计条：全部 / 按时 / 超时 / 延期 / 待递交（⚠️ 纯统计数字，**不可点击筛选**）
3. 「最新/最早」排序可用（首卡日期随之切换）
4. 空项目显示「该项目暂无递交记录」

## 可直调接口

- `project_publish/get_list?project_id=N`
  - ⚠️ GET 直调时 `project_id` **不参与过滤**（任意 id 返全量约 12810 条）——页面内实际取数走另一通道，接口回归断言不要依赖该参数
  - `keywords` 按项目名精确匹配

## 已知坑

- ⚠️ 顶部数字块纯统计不可点筛选
- ⚠️ 接口 `project_id` GET 直调不过滤（见上）
